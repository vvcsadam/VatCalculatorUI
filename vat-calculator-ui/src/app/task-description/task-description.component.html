GLOBAL BLUE - HOMEWORK

User Story

As a web portal user, I would like to calculate Net, Gross, and VAT amounts for purchases in Austria so that I can use correctly calculated data.

Acceptance Criteria

  • If the user inputs one of the net, gross, or VAT amounts and additionally a valid Austrian VAT rate (10%, 13%, 20%), the other two missing amounts (net/gross/VAT) are calculated by the system and shown.
  • The system provides an error with meaningful error messages in case of:
    • Missing or invalid (0 or non-numeric) amount input, or more than one input.
    • Missing or invalid (0 or non-numeric) VAT input.
  • The web interface must be responsive and needs to work on all standard desktop resolutions â€“ supported browsers are Mozilla, Chrome, Edge (no mobile or tablet support).

Technical Requirements

  • The solution needs to be implemented in the Angular 18 framework.
  • The solution needs to use the Angular Material CSS framework/UI components (Angular Material).
  • The backend implementation shall be done with the .NET 8.0 framework and utilize the dependency injection (DI) design pattern and SOLID principles.
  • The API needs to fulfill the REST API standards.
  • The solution needs to be published in a public GitHub repository accessible by the GB team.
